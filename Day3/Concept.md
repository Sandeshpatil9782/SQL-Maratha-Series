# Day 3/16: USE DATABASE ⚔️ | SQL Swarajya Goshti

## 📜 Concept: राज्यात प्रवेश, कारभार सुरू

काल आपण गुप्तचरांकडून सगळ्या राज्यांची यादी मागवली. `SHOW DATABASES` करून खात्री केली की आपलं `Swarajya` अस्तित्वात आहे.

पण यादी बघून स्वराज्य चालत नाही. **आत जाऊन गादीवर बसावं लागतं. तेव्हाच खरा कारभार सुरू होतो.**

छत्रपती शिवाजी महाराजांनी रायगड जिंकला. पण ते फक्त जिंकून थांबले नाहीत. त्यांनी रायगडावर जाऊन राजधानी केली, सिंहासन स्थापन केलं. **गडात प्रवेश केल्यावरच स्वराज्याचे हुकूम सुटू लागले.**

SQL मध्ये `USE Swarajya;` हा तोच क्षण आहे. रिकाम्या वाड्यात प्रवेश करून आता आपण इथे खाती-वह्या मांडणार आहोत.

## 💻 SQL Command: `USE DATABASE`

### **1. Syntax**
```sql
USE database_name;
