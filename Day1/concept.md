# Day 1: CREATE DATABASE ⚔️ | 

## स्वराज्याची सुरुवात

प्रत्येक गोष्टीची सुरुवात कुठेतरी होते. छत्रपती शिवाजी महाराजांनी जिजाऊ आऊसाहेबांसोबत लाल महालात बसून स्वराज्याचा संकल्प केला. रयतेचं राज्य निर्माण करण्याची शपथ घेतली. 
**तीच होती स्वराज्याची पहिली `CREATE` command.**

संकल्पाशिवाय कृती नाही, आणि कृतीशिवाय स्वराज्य नाही.

## SQL मध्ये `DATABASE` म्हणजे काय?

`DATABASE` म्हणजेच तुझं स्वतःचं स्वराज्य. एक रिकामी जागा, एक रिकामा वाडा. 

जसं शिवरायांनी आधी राज्य स्थापन केलं, मग त्यात किल्ले, मावळे, खजिना आला. तसंच SQL मध्ये आधी `DATABASE` बनवावा लागतो. मग त्यात `TABLES` येतात, `DATA` येतो.

`CREATE DATABASE Swarajya;` हा आदेश म्हणजे तू तुझ्या कामासाठी एक नवीन वही उघडतोयस.

## पाया महत्वाचा का?

शिवाजी महाराजांनी गड-किल्ले बांधताना पाया भक्कम केला. म्हणून 350 वर्षांनी सुद्धा रायगड उभा आहे.

Database हाच तुझ्या पूर्ण प्रोजेक्टचा पाया आहे. 
- पाया कच्चा असेल तर इमारत कोसळते.
- Database चं नाव चुकीचं, रचना चुकीची असेल तर पुढचा सगळा प्रोजेक्ट गोंधळात पडतो.

**म्हणून नोंद अशी करा की भविष्याला उपयोगी पडेल।**

## आजचा संकल्प

आज आपण फक्त `CREATE` करायला शिकलो. कृतीची सुरुवात झाली. 
उद्या आपण ह्या `Swarajya` मध्ये प्रवेश करणार आहोत `USE` command ने.
